## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript for Outsera Test Project.

## Project installation

```bash
$ yarn install
```

## Compile and run the project

```bash

# development
$ yarn start

```

## Run tests

```bash

# development
$ yarn test

```
